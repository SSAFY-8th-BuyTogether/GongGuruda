# 댓글 보기
## 기능


### ✔️ 댓글 보기
댓글 작성자, 작성자 프로필, 댓글 내용, 작성일자를 확인할 수 있음


### ✔️댓글 작성
댓글 내용을 작성할 수 있음


### ✔️댓글 삭제
본인이 작성한 댓글에는 메뉴 버튼이 활성화되고 해당 버튼을 클릭하여 댓글 삭제가 가능함


### ✔️맨션
본인 댓글이 아닌 댓글을 클릭시 맨션 레이아웃을 활성화함  
맨션된 사용자는 파란색으로 표시되고 알람이 감
