# 알람 
## 기능


### ✔️알림 보내기
알림을 보내는 조건이 생기면 서버로 알림 내용과 받는 사람 정보를 보내고  
서버는 firebase로 FCM을 요청함


### ✔️알림 받기
내가 작성한 글에 참여자가 발생하거나 댓글이 달렸을 경우  
댓글에서 내가 맨션이 되는 경우 알림이 발생함
